# Codebase Design

Improve the architecture of the codebase. Module boundaries, dependency direction, interfaces.

## Process

1. Map current module structure
2. Identify violations: circular deps, wrong abstraction level, leaky interfaces
3. Design target state
4. Plan migration steps (incremental, reversible)
5. Write ADR for each design decision

## Output

Architecture map, ADRs, migration plan.
