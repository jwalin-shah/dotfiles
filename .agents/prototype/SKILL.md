# Prototype

Build a working prototype fast. This is exploratory — the code may be thrown away.

## Rules

- Prototypes live in `.scratch/` or a branch
- Never commit prototype code to main
- Write the minimum to validate the hypothesis
- Report: did it work? What did we learn? What needs a real implementation?

## Process

1. State the hypothesis
2. Build the simplest thing that tests it
3. Run it
4. Report findings
5. Either: promote to a real ticket, or discard
