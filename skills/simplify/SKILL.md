# Simplify Skill
When the user uses `/simplify`, your sole objective is to reduce complexity.
1. Analyze the target code.
2. Inline unnecessary functions, remove dead code, and delete over-engineered abstractions.
3. The end result MUST have fewer lines of code and be easier to read.
4. Do NOT add new features.
